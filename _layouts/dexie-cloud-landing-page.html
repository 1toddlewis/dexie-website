Get Started

This guide assumes you're familiar with using npm to manage dependencies. If you're unfamiliar with all that, try read webpack's get started guide.

1. Install dexie and dexie-cloud

npm install dexie dexie-cloud

2. Create your database in the cloud

2. Create your db in the cloud

npx dexie-cloud create-db

3. Whitelist your app URLs

npx dexie-cloud whitelist <URL>

4. Code your app

  import Dexie from "dexie";
  import DexieCloud from "dexie-cloud";

  // Declare your DB
  const db = new Dexie('myDB', {addons: [DexieCloud]});
  db.version(1).stores({
    todoItems: '$$id, title, done' // $$ = autogenerated UUID
  });

  // Connect your dexie-cloud database:
  db.cloud.connect("https://<yourdatabase>.dexie.cloud", {
    requireAuth: true
  });

5. Done!

Enjoy.

With this setup, users will have to login using email OTP the first time they go to your app. If you already have another authentication in place, or want to integrate with the authentication of your choice, this can be defined in the options object passed to db.cloud.connect(). The API docs for Dexie Cloud will provide more details when released...

Your app will now instantly sync changes with the cloud. If offline, changes are queued until next time you are online.

Your app can now use Dexie as you normally would do in Dexie.js - add, update, delete and query db.todoItems.
