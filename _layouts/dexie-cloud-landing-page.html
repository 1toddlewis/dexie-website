Get started with Dexie Cloud

This guide assumes you're familiar with using npm to manage dependencies. If you're unfamiliar with all that, try read webpack's get started guide.

1. Install dexie and dexie-cloud

cd to your web app and install dexie and dexie-cloud:

npm install dexie dexie-cloud

2. Create your database in the cloud

npx dexie-cloud create-db

This command will prompt for and verify your email address and then create a new database for you in the cloud.

Your database will be given its own unique URL, printed out on the console. Use that URL in step 3.

Two new files will also be saved locally. The files points out the db URL and authenticates further usage of the dexie-cloud CLI. The files are not required for the runtime webapp.

~/my-dexie-app $ npx dexie-cloud create-db
Enter your E-mail address: foo@bar.com
Enter OTP (check your email!): x067ae
Database successfully created.
Database URL: https://dnsdjknfd.dexie.cloud
~/my-dexie-app $ ls -l
-rw-r--r--    1 user staff       4  1 Oct 15:38 dexie-cloud.json
-rw-r--r--    1 user staff       4  1 Oct 15:38 dexie-cloud.secret
...

The json file may be added to git, but never the .secret one.

3. Code your app

  import Dexie from "dexie";
  import DexieCloud from "dexie-cloud";

  // Declare your DB
  const db = new Dexie('myDB', {addons: [DexieCloud]});
  db.version(1).stores({
    todoItems: '$$id, title, done' // $$ = autogenerated UUID
  });

  // Connect your dexie-cloud database:
  db.cloud.connect("https://<yourdatabase>.dexie.cloud", {
    requireAuth: true
  });

  document.getElementById('addButton').onclick = async () => {
    // Create data:
    await db.todoItems.add({title: "Foo" + Math.random(), done: 0});
  }

  // Query data:
  db.todoItems.where({done: 0}).toArray().then(todoItems => {
    console.log(todoItems);
  });

4. Done!

Refresh webpage and enjoy bidirectional sync!
